# Comandos para empezar crear un Backend
```node
// Ver la versión actual de node
node --version

// Ver la versión actual de npm
npm --version

// Inicializar un nuevo proyecto de Node.js
npm init -y

// Instalación dependencias para el backend
npm install express body-parser cors

// Para inicializar más rápido la app en el package.json poner en el apartado de scripts:
"start": "node index.js"
// Ahora en la terminal solo debemos poner npm start

// Validar que el .gitignore no suba los node_modules

```
